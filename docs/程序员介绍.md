## 程序员分类

### 1. **前端开发**

- 做浏览器页面、网页交互、UI 界面
- 技术：HTML / CSS / JavaScript / Vue / React
- 特点：视觉 + 交互

### 2. **后端开发**

- 负责系统逻辑、接口、数据库、业务流程
- 技术：Java / Go / Python / Node.js
- 特点：业务复杂、系统架构性强

### 3. **全栈开发**

- 前后端都能干
- 特点：小团队和创业公司最需要

### 4. **移动端开发**

- 安卓、iOS App
- 技术：Kotlin / Swift / Flutter / React Native

### 5. **桌面软件开发**

- Windows、macOS 应用程序
- 技术：C#、C++、Qt、Electron

### 6. **嵌入式开发**

- 单片机、物联网、硬件控制
- 技术：C、C++、RTOS、芯片SDK（如 ESP-IDF、STM32）
- 特点：和硬件紧密相关

### 7. **游戏开发**

- 3D、2D 游戏制作
- 技术：Unity / Unreal / C++ / C#

### 8. **数据方向**

- **数据分析**（Python、SQL）
- **数据工程**（ETL、数据平台）
- **算法工程师**（机器学习、深度学习）
- 特点：数学要求高（尤其算法）

## 如何学习

首先，广告做的最多的就是培训机构，也就是python速成班之类的。但是程序是一种考开源发展进步的领域，我们可以在网上找到免费的教程。

首先，我们要学会在b站上找到免费优质的教程。我们想要学什么语言，直接在b站上搜就有了，大部分都简单易懂。

我们不能只看视频，那样子是学不会的。我们需要根据视频进行实践。做出一个个的项目，这样我们才能有所进步。

大部分的开源仓库都开源在了github，我们需要学会稳定访问github。

git是代码的版本管理，我们可以在网上学习如何使用git，来开源管理我们的代码。

学习中遇到了问题怎么办？

我们可以查看错误信息，并复制到搜索引擎中，比如必应和谷歌。虽然csdn有一堆广告，但也有很多解决报错的文章。

我们也可以问大佬，需要将问题清晰的描述出来，才有大佬为你解答。

看视频学习更适合某个领域的入门，大部分的资料都是以文档的形式出现的。

某个开源项目会以网页的形式进行展示。

我们需要有看懂文档的能力， 并根据文档搭建开源项目。

如果有些文档过于简略，我们仍可以在搜索引擎找到教程。
