# AI8051U硬件介绍

开发板装在塑料盒子里，下面有usb线和短接帽。

实验箱有很多接口，

- 立体声输出
- 立体耳机输出
- 话筒录音
- OLED spi/iic显示屏
- 8路流水灯
- 8路数码管
- 并口TFT彩屏
- 掉电检测电压调节
- 红外接收
- 红外发射
- 矩阵键盘
- T0 T1 外部中断按键
- INT0、INT1按键
- TF卡插座
- USB转双串口
- USB Link1D接口
- USB_TypeA
- RTC电池
- 外部并行总线
- QSPI/SPI Flash

## 环境配置

### 安装keil5编程软件

在keil官网，下载c251文件，然后安装。安装目录不能有中文。

### 获取芯片数据手册

我们访问[stc官网](stcai.com)，选择芯片数据手册，选择AI8051U，下载规格书。

在软件工具里，选择工具软件，下载Alapp-ISP。

我们可以在软件工具的实验箱里下载AI88051U的资料。

### 添加型号和头文件到keil中

在alapp-ISP软件中，在右侧工具栏选择keil仿真设置，点击`添加型号和头文件到keil中`，选择keil_v5文件夹，在此文件夹有c251文件夹。即可添加成功。

### 下载插件

在stcai.com 工具软件中，下载keil中断拓展工具，解压工具，选择keil5的安装目录，即可添加成功。
